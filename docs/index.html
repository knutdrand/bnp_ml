Installation¶

Stable release¶

To install BioNumPy Machine Learning, run this command in your terminal:

$ pip install bnp_ml

or for the newest:

$ pip install git://github.com/knutdrand/bnp_ml

If you don’t have pip installed, this Python installation guide can guide you through the process.

From sources¶

The sources for BioNumPy Machine Learning can be downloaded from the Github repo.

You can either clone the public repository:

$ git clone git://github.com/knutdrand/bnp_ml

Once you have a copy of the source, you can install it with:

$ pip install -e .

Usage¶

To use BioNumPy Machine Learning in a project:

import bnp_ml
